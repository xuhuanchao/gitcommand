intellJ idea快捷键


mac os

1.  command + [ 或 ]，可以跑到大括号的开头与结尾
2.  command + p 显示参数信息
3.  command + i 实现接口方法
4.  command + u 进入父类
5.  command + n 找类
6.  control + return  生成get set
    control + n 自动生成提示
7.  command + f12 查方法列表
8.  control + shift + f   在java path中查寻
9.  option + command + left/right 前进后退页面
10. option + command + b    进入实现方法
11. command + b 进入接口方法
12. command + z / command + shift + z   撤销／反撤销
